# Flask Task Manager

A simple task manager web application 
built with Flask and SQLite.

## Features
- Add tasks
- Delete tasks
- Mark tasks as complete
- Persistent storage with SQLite

## Tech Stack
- Python
- Flask
- Flask-SQLAlchemy
- HTML/CSS
