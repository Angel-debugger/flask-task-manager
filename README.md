# Flask Task Manager

A simple task manager web application built with Flask and SQLite.

## Features
- Add tasks
- Delete tasks
- Mark tasks as complete
- Persistent storage with SQLite

## Tech Stack
- Python
- Flask
- Flask-SQLAlchemy
- HTML/CSS

## How to Run Locally

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
